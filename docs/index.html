Spiral

This spiral is a very close approximation to the Fibonacci Golden Spiral. It uses a growth factor, b, of 0.005347.

Sunflower

Sunflower seeds are arranged according to the golden ratio. We simulate this with this workspace. Read this article for more details.

Instructor Action
